This starter ships with a full favicon set so the site works out of the box, but you should replace it with your own identity.

## Where the Files Live

Favicons are stored under:

```text
assets/img/favicons/
```

You can replace the existing files directly if you want to keep the current file names and workflow unchanged.

## What To Prepare

- a square favicon
- an Apple touch icon
- an SVG favicon if your design supports it

## Why This Matters

Even a small writing site feels more intentional once browser tabs, mobile homescreen icons, and pinned shortcuts all use the same visual identity.
