# Upstream Notice

This repository is a small customization of the official Chirpy ecosystem. It
uses `jekyll-theme-chirpy` as the theme layer and was bootstrapped from the
official `chirpy-starter` template.

This notice stays in the repository for attribution and is excluded from the
generated site.

## What This Repository Changes

This repository keeps the Chirpy gem-based structure and adds local starter
overrides for:

- editorial typography and color styling
- self-hosted fonts and preload hooks
- sidebar and topbar customizations
- archive layout adjustments
- example posts, placeholder content, and starter assets

When upgrading Chirpy, review those local overrides against upstream changes.
